| module | 模組 | toml 檔 |
| package | 套件: 由一系列相關功能的 crate 所組成的單元 | 內含一個 Cargo.toml 檔，用來說明如何建構 crates |

| library crate | 函式庫 Crate: 用來定義分可享工具的單元 | 不具 main 函式 |
| crate root | Crate 源頭: Rust compiler 組織 Modules 的起始 Crate |  |
