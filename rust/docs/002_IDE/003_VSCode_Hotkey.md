# VS Code 快速鍵

| Description | hot key |
|----------|-----------------|
| 行複製 | Shift + Alt + Up/Down |
| line move up/down | Alt + up/down |
| variable rename | F2 |
| code formatting | Shift + Alt + F |
| delete line | Ctrl + x |
| jump to implementation | F12 |
